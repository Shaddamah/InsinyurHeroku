__[Found bug? Let us know!](https://github.com/Shaddamah/irmartabak-heroku/issues)__


## Development Setup

__Setup__

```
npm install
pip install requirements.txt
```

__Bundle Webpack__

```
npm run build
```

__Running Server__

```
2 Ways:
1. open terminal then run server.bat
2. npm start
```
